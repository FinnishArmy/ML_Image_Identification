This is a simple image identifcaion AI that detects certain objects from an image from
the word data base. More words may be added, but the PB files must also be updated accordingly.


It is a must to install OpenCV.

This code was created in PyCharm with the 'opencv-python' python package installed. 

Steps: 
File -> Settings -> Project -> Python Interpreter -> Python 3.9 -> + -> opencv-python
Then you should be able to run the code properly.
